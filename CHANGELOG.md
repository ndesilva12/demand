# Changelog

## [1.0.0] - 2026-02-16

### Initial Release - MVP

**Features:**
- User authentication (Email/Password + Google)
- Create demands with measurable success criteria
- Browse and filter demands (active, won, closed)
- Co-sign demands (with toggle)
- User dashboard (my demands + co-signed demands)
- Responsive design (mobile-friendly)
- Real-time Firestore integration

**Tech Stack:**
- Next.js 15 (App Router)
- Firebase (Auth + Firestore)
- TypeScript
- Tailwind CSS

**Deployment:**
- GitHub: https://github.com/ndesilva12/demand
- Vercel: Auto-deploy enabled

**Coming Soon:**
- Message boards per demand
- Democratic editing (vote on changes)
- Company research tool
- Email notifications
- Social sharing
- Search and advanced filters
